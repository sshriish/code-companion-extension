Explanation

This function walks the token list once, pushing onto a stack whenever it sees an opening bracket and popping to check for a match on a closing one.

Complexity

Time: O(n) Space: O(n)

Single pass over the input; the stack can hold up to n tokens.

Likely interview questions

  • How would you validate multiple bracket types at once?
  • How would you do this with O(1) extra space?
